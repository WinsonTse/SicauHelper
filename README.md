#川农校园通
> 川农校园通是我的毕业设计，只需要使用川农的学号和密码，就可以直接使用所有的功能，
> 数据主要是从教务系 统上扒下来的。当然，也是很安全的。有兴趣的同学可以安装试试，
> 需要 Android 4.0 及以上版本的手机。

[直接下载][1]
[百度网盘][2]

**主要功能**

* 理论课表
* 实验课表
* 我的成绩
* 教务新闻
* 空闲教室
* 校内市场
* 川农圈子
